# Job Portal Web Application

A **Job Portal Web Application** that connects **job seekers** and **recruiters** on one platform.  
Job seekers can search and apply for jobs, while recruiters can post job openings and manage applications.

---

## Project Overview

This project is developed as a **college project** to understand how real-world job portals work.  
It demonstrates the basic workflow of recruitment systems used in modern web applications.

---

## Features

- User Registration & Login
- Job Seeker and Recruiter roles
- Job posting by recruiters
- Job search and job listing
- Apply for jobs
- Simple and user-friendly interface
- Secure data handling
